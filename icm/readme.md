### Identity Card Management  

*Demo*
[Youtube](https://youtu.be/OLgYX0qDSE4)

*Requirement*
[Ganache](https://truffleframework.com/ganache)
[Metamask](https://metamask.io/)
[Truffle framework](https://truffleframework.com/)

*Build contract*  
```bash
cd icm
truffle compile
```

*Migrate contract*  
```bash
truffle migrate
```

*Install node module*  
```bash
cd app
npm install
```  

*Start webapp*  
```bash
node app.js
```

*Webapp run on port 9000*
